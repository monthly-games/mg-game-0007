## 3. 1년차 라이브 플랜

### 3.1 초기 (1~3개월차)

| 주차 | 주요 활동 |
|------|----------|
| 1~2주 | 소프트 런칭, 버그 수정 |
| 3~6주 | 아침/저녁 루틴 구조 설계 및 튜토 강화 |
| 7~10주 | 농장 테마 2~3종 우선 제공 |
| 11~12주 | 봄 축제 준비 |

**핵심 과제:**
- 루틴 시스템 튜토리얼 강화
- 초반 농장 성장 체감 최적화
- 테마 다양성 확보

### 3.2 중기 (4~8개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 4개월 | 봄 꽃축제 이벤트 |
| 5개월 | 마을/NPC 서브퀘스트 추가 |
| 6개월 | 여름 수확제 이벤트 |
| 7개월 | 가을 수확제 이벤트 |
| 8개월 | NPC 스토리라인 확장 |

**핵심 과제:**
- 계절 이벤트 사이클 정착
- 마을 콘텐츠 확장
- NPC 관계 시스템 강화

### 3.3 후기 (9~12개월차)

| 월 | 주요 업데이트 |
|----|-------------|
| 9개월 | 다른 타이틀과 자원/이벤트 연동 실험 |
| 10개월 | 힐링 굿즈/브랜드 콜라보 탐색 |
| 11개월 | 겨울 축제 이벤트 |
| 12개월 | 1주년 이벤트 + 연말 테마 |

**핵심 과제:**
- 크로스 게임 연동 검증
- 브랜드 콜라보 가능성 검증
- 장기 유저 리텐션
