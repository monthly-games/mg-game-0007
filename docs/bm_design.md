## 3. 상세 BM 설계

### 3.1 꾸미기/인테리어 BM

#### 무료 구조
- 스토리 진행으로 기본 장식 30종 획득
- 계절 이벤트에서 한정 데코 파밍
- 업적 달성으로 특수 장식 해금

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 테마 패키지 | 농장 테마 세트 (건물+장식) | $4.99~$9.99 |
| 단품 장식 | 개별 프리미엄 장식 | $0.99~$2.99 |
| 시즌 한정 세트 | 봄/여름/가을/겨울 테마 | $6.99 |
| 동물 스킨 | 동물 외형 변경 | $1.99~$3.99 |
| 농장 배경 | 전체 배경 변경 | $2.99 |

### 3.2 방치 수익 BM

#### 무료 구조
- 오프라인 수확 상한: 8시간
- 자동 판매 수익 누적
- 일일 접속 보너스

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 수확 2배 | 광고 시청 시 적용 | 무료 (광고) |
| 수확 3배 패스 | 상시 적용 (30일) | $2.99/월 |
| 오프라인 확장 | 16시간까지 누적 | 패스 포함 |
| 자동 수확기 | 작물 자동 수확 | $1.99 (영구) |

### 3.3 농장 확장 BM

#### 무료 구조
- 레벨업으로 농장 면적 자동 확장
- 퀘스트 완료로 시설 해금
- 마을 발전도 무료 진행

#### 유료 레버
| 항목 | 설명 | 가격대 |
|------|------|--------|
| 즉시 확장권 | 다음 확장 즉시 해금 | $2.99~$4.99 |
| 프리미엄 시설 | 특수 건물 (온실, 양조장 등) | $4.99 |
| 농장 슬롯 | 추가 농장 운영 | $3.99 |
| 동물 슬롯 | 동물 사육 +5 | $1.99 |

### 3.4 광고 BM

| 광고 유형 | 적용 | 보상 |
|----------|------|------|
| 리워드 광고 | O | 수확 2배, 성장 가속, 희귀 씨앗 |
| 인터스티셜 | X | - |
| 광고 제거 패스 | O | $1.99 (영구) |

## 6. BM 밸런스 원칙

### P2W 방지선
- 모든 콘텐츠 무과금으로 접근 가능 (시간 차이만)
- 꾸미기 아이템 기능성 버프는 1~3% 수준
- 과금 농장과 무과금 농장 생산성 격차 최소화

### 과금 압박 최소화
- 힐링 게임 특성상 경쟁/압박 요소 없음
- 루틴 플레이만으로 충분한 진행
- 광고도 완전 선택적
