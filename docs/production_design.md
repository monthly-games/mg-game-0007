## 1. 재미 요소 (Fun Factors)

### 1.1 핵심 재미 (Core Fun)
| 요소 | 설명 | 강도 |
|------|------|------|
| **반사 신경** | 빠른 반응으로 장애물 회피 | ★★★★★ |
| **기록 경쟁** | 최고 점수/거리 갱신의 성취감 | ★★★★★ |
| **단순함** | 쉽게 배우고 바로 즐기는 접근성 | ★★★★☆ |
| **긴장감** | 점점 빨라지는 속도의 스릴 | ★★★★☆ |

### 1.2 만족감 요소 (Satisfaction)
- **회피 성공**: 아슬아슬하게 장애물 회피 시 쾌감
- **콤보 유지**: 연속 장애물 회피 콤보
- **신기록**: 개인/친구 기록 갱신
- **보상 획득**: 코인/아이템 수집

### 1.3 도전 요소 (Challenge)
```
도전 구조:
├── 점점 빨라지는 게임 속도
├── 다양한 장애물 패턴
├── 특수 구간 (점프대, 슬라이드존)
├── 보스 구간 (특수 이벤트)
└── 일일 챌린지 (특수 조건)
```

### 1.4 사회적 요소 (Social)
- **리더보드**: 글로벌/친구 점수 랭킹
- **고스트 레이스**: 친구 기록과 함께 달리기
- **공유**: 기록 SNS 공유
- **일일 챌린지**: 동일 조건 경쟁

## 2. 게임 루프 (Game Loop)

### 2.1 코어 루프 (30초~3분)
```
┌─────────────────────────────────────────────────┐
│                    코어 루프                      │
├─────────────────────────────────────────────────┤
│  [달리기 시작] → [장애물 회피] → [아이템 수집]    │
│       ↑                              ↓           │
│       └──── [게임 오버] ← [충돌] ←───┘           │
└─────────────────────────────────────────────────┘
```

**단계별 상세:**
1. **달리기 시작** (1초): 탭하여 게임 시작
2. **장애물 회피** (지속): 점프/슬라이드로 장애물 회피
3. **아이템 수집** (지속): 코인, 부스터 아이템 수집
4. **충돌** (즉시): 장애물 충돌 시 게임 오버
5. **게임 오버** (5초): 결과 확인, 보상 수령

### 2.2 세션 루프 (5~15분)
```
[접속] → [일일 보상] → [런 시작] → [게임 오버] → [보상 확인]
                             ↓                        ↓
                        [재도전]              [캐릭터 강화]
```

**세션 구성:**
- 일일 보상: 10초
- 런 3~5회: 3~10분
- 캐릭터/업그레이드 확인: 2~3분

### 2.3 메타 루프 (장기)
```
┌──────────────────────────────────────────────────────┐
│                     메타 진행                         │
├──────────────────────────────────────────────────────┤
│  [코인 수집] → [캐릭터 해금] → [업그레이드 강화]       │
│         ↓                              ↓             │
│  [신기록 도전]                   [프레스티지 보너스]   │
│         ↓                              ↓             │
│  [더 먼 거리 도전] ←─────────────────────┘           │
└──────────────────────────────────────────────────────┘
```

### 2.4 플로우 다이어그램
```
[앱 시작]
    ↓
[메인 메뉴] ────→ [캐릭터 선택]
    │             [상점]
    │             [리더보드]
    │             [설정]
    ↓
[PLAY] → [게임 화면] → [달리기]
                          ↓
                     [게임 오버]
                          ↓
                     [결과/재도전]
```

## 3. BM 모델 (Business Model)

### 3.1 수익화 전략
| 채널 | 비중 | 설명 |
|------|------|------|
| **리워드 광고** | 55% | 부활, 보상 2배, 부스터 |
| **캐릭터 스킨** | 20% | 코스메틱 캐릭터 |
| **편의 기능** | 15% | 광고 제거, 코인 부스터 |
| **인터스티셜** | 10% | 게임 오버 후 광고 |

### 3.2 과금 포인트
```
비과금 유저 경험:
├── 모든 게임 모드 플레이 가능
├── 기본 캐릭터 사용
├── 광고 시청으로 부활/부스터
└── 일일 무료 코인 보너스

과금 유저 혜택:
├── 광고 제거 ($2.99)
├── 캐릭터 스킨 ($0.99~$2.99)
├── 코인 패키지 ($0.99~$9.99)
├── 스타터 팩 ($3.99)
└── VIP 패스 ($4.99/월)
```

### 3.3 가격 정책
| 상품 | 가격 | 가치 |
|------|------|------|
| 소량 코인 (1000) | $0.99 | 캐릭터 해금용 |
| 중량 코인 (5000) | $2.99 | 20% 보너스 |
| 대량 코인 (12000) | $4.99 | 40% 보너스 |
| 광고 제거 | $2.99 | 영구 적용 |
| 캐릭터 팩 | $0.99~2.99 | 개별 캐릭터 |
| 스타터 팩 | $3.99 | 코인 3000 + 캐릭터 + 광고 제거 |

### 3.4 리텐션 메커니즘
- **일일 보상**: 출석 체크 (7일 사이클)
- **일일 퀘스트**: 거리 달성, 코인 수집
- **주간 챌린지**: 특수 조건 달성
- **시즌 이벤트**: 테마 스킨, 특별 모드

## 4. 레벨 디자인 (Level Design)

### 4.1 난이도 곡선
```
속도
  ▲
  │                              ╭──────
  │                         ╭────╯
  │                    ╭────╯
  │               ╭────╯
  │          ╭────╯
  │     ╭────╯
  │╭────╯
  └──────────────────────────────────────→ 거리
    0   500  1000 1500 2000 2500 3000
```

**난이도 공식:**
```
게임_속도 = 기본_속도 × (1 + 거리/1000 × 0.1)
최대_속도 = 기본_속도 × 2.0
장애물_빈도 = 기본_빈도 × (1 + 거리/500 × 0.15)
```

### 4.2 콘텐츠 진행
| 구간 | 거리 | 새로운 요소 |
|------|------|-------------|
| 튜토리얼 | 0~100 | 기본 조작, 단순 장애물 |
| 초반 | 100~500 | 코인, 다양한 장애물 |
| 중반 | 500~1500 | 부스터, 점프대 |
| 후반 | 1500~3000 | 복합 패턴, 슬라이드존 |
| 고난이도 | 3000+ | 극한 속도, 랜덤 패턴 |

### 4.3 밸런싱 공식
```dart
// 점수 계산
int calculateScore(int distance, int coins, int combo) {
  int baseScore = distance;
  int coinBonus = coins * 10;
  int comboBonus = combo * 5;
  return baseScore + coinBonus + comboBonus;
}

// 코인 획득
int coinsPerRun(int distance) {
  return distance ~/ 10; // 10m당 1코인
}

// 업그레이드 효과
double upgradeBonus(String upgradeId, int level) {
  switch (upgradeId) {
    case 'magnet': return level * 0.5; // 범위 증가
    case 'shield': return level * 1.0; // 지속시간 증가
    case 'coin_bonus': return level * 0.1; // 코인 10% 증가
  }
}
```

### 4.4 보상 체계
| 보상 유형 | 획득 조건 | 수량 |
|-----------|----------|------|
| 런 코인 | 수집 | 직접 수집량 |
| 거리 보너스 | 거리 달성 | 거리/100 코인 |
| 일일 보너스 | 출석 | 100~500 코인 |
| 퀘스트 보상 | 완료 | 200~1000 코인 |
| 업적 보석 | 마일스톤 | 10~100 보석 |

## 5. UI/UX 설계

### 5.1 정보 계층
```
레벨 1 (항상 표시):
├── 현재 점수/거리
├── 코인 수
├── 활성 부스터
└── 일시정지 버튼

레벨 2 (상호작용 시):
├── 게임 오버 화면
├── 부스터 선택
└── 부활 옵션

레벨 3 (팝업):
├── 결과 화면
├── 신기록 연출
├── 설정 메뉴
└── 상점
```

### 5.2 네비게이션 흐름
```
[메인 메뉴]
    ├── [PLAY] → [게임] → [게임 오버] → [결과]
    ├── [캐릭터] → [캐릭터 선택/해금]
    ├── [상점] → [코인/보석/특별 상품]
    ├── [리더보드] → [랭킹]
    └── [설정] → [옵션]
```

### 5.3 피드백 시스템
| 이벤트 | 시각 | 청각 | 촉각 |
|--------|------|------|------|
| 점프 | 점프 애니메이션 | 점프음 | - |
| 슬라이드 | 슬라이드 애니메이션 | 슬라이드음 | - |
| 코인 획득 | 코인 이펙트 | 코인음 | - |
| 아슬아슬 회피 | 근접 이펙트 | 근접음 | 진동 (약) |
| 충돌 | 폭발 이펙트 | 충돌음 | 진동 (강) |
| 신기록 | 신기록 배너 | 팡파레 | 진동 (중) |

### 5.4 접근성
- **원터치 조작**: 점프/슬라이드 단순 조작
- **속도 조절**: 초보자 모드 (느린 시작)
- **색맹 모드**: 장애물 구분 패턴
- **효과 감소**: 시각 효과 줄이기
- **세로 모드**: 한 손 플레이 최적화
