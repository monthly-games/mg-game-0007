# MG-0007: Pixel Farm UI/UX 스펙

> **게임명**: Pixel Farm (픽셀 농장)
> **장르**: 농장 경영 시뮬레이션
> **카테고리**: Year 1 Core
> **문서 버전**: 1.0.0

---

## 1. 화면 설정

| 항목 | 값 |
|------|-----|
| **방향** | Portrait (세로) |
| **기준 해상도** | 1080 × 1920 |
| **지원 비율** | 16:9 ~ 21:9 |
| **타겟 FPS** | 60fps |

### 방향 선택 이유
- 농장 맵 세로 스크롤
- 탭으로 수확 최적화
- 경영 UI 하단 배치

## 2. Safe Area

| 영역 | 값 | 용도 |
|------|-----|------|
| 상단 | 44-62dp | 자원, 레벨 |
| 하단 | 34-48dp | 도구, 네비게이션 |

## 4. 터치 영역

| 동작 | 영역 | 제스처 |
|------|------|--------|
| 작물 심기 | 셀 영역 | 탭 |
| 수확 | 작물 위 | 탭 |
| 전체 수확 | 버튼 | 탭 |
| 맵 스크롤 | 맵 영역 | 스와이프/드래그 |
| 건물 진입 | 건물 위 | 탭 |
| 핀치 줌 | 맵 영역 | 핀치 |

### 터치 피드백
- 심기: 씨앗 뿌리기 애니메이션
- 수확: 작물 팝 + 자원 플라이
- 물주기: 물방울 이펙트
