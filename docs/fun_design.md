## 2. 코어 루프 (Core Loop)

```
┌─────────────────────────────────────────────────────────┐
│  1. 농장 관리 (수확, 씨앗 심기, 동물 돌보기)              │
│         ↓                                               │
│  2. 자원 수집 후 마을/모험 준비                          │
│         ↓                                               │
│  3. 간단한 전투/퀘스트 진행                              │
│         ↓                                               │
│  4. 농장/마을 업그레이드 및 꾸미기 ─────────→ (1로 반복)  │
└─────────────────────────────────────────────────────────┘
```

### 루프 상세 설명

1. **농장 관리**: 작물 심기/수확, 동물 돌보기, 시설 가동
2. **자원 수집**: 농장 산출물로 모험 준비물 제작
3. **전투/퀘스트**: 가벼운 RPG 요소로 스토리 진행
4. **업그레이드**: 농장 확장, 마을 발전, 외형 꾸미기

## 3. 재미 축 (Fun Pillars)

### 3.1 힐링/루틴 (Healing)
- **목표**: 편안하고 스트레스 없는 플레이 경험
- **핵심 경험**: 하루의 피로를 풀어주는 힐링 타임
- **주요 요소**:
  - 아기자기한 픽셀 아트
  - 부드러운 BGM
  - 느긋한 게임 페이스
  - 실패 없는 시스템

### 3.2 성장/확장 (Growth)
- **목표**: 농장이 점점 커지고 풍요로워지는 경험
- **핵심 경험**: 처음엔 작은 밭이 대농장이 되는 성취감
- **주요 요소**:
  - 농장 면적 확장
  - 작물/동물 종류 증가
  - 시설 업그레이드
  - 자동화 시스템

### 3.3 꾸미기 (Expression)
- **목표**: 나만의 개성 있는 농장 만들기
- **핵심 경험**: 내 농장에 대한 애착과 자부심
- **주요 요소**:
  - 농장 레이아웃 자유도
  - 장식 아이템
  - 시즌 테마
  - 친구 방문 시스템

## 4. 디자인 레버 (Design Levers)

### 4.1 루틴 체크리스트
| 레버 | 설명 |
|------|------|
| 아침/저녁 루틴 | 하루 5~10분이면 끝나는 "아침/저녁 루틴" 체크리스트 설계 |
| 구현 방향 | 아침: 수확→판매→씨앗심기, 저녁: 동물돌봄→퀘스트확인→잠자기 |

### 4.2 루프 연결 구조
| 레버 | 설명 |
|------|------|
| 자원 상호 보강 | 농장 자원 ↔ 모험 보상 간 상호 보강 구조로 루프 연결 |
| 구현 방향 | 농장 음식 → 모험 버프, 모험 보상 → 희귀 씨앗/동물 |

### 4.3 NPC 생활감
| 레버 | 설명 |
|------|------|
| 시간/날씨 기반 루틴 | NPC들의 시간/날씨 기반 루틴으로 생활감 제공 |
| 구현 방향 | 아침엔 빵집 NPC 배달, 비오는 날 낚시꾼 등장, 계절별 축제 |

### 4.4 꾸미기 기능성
| 레버 | 설명 |
|------|------|
| 소소한 버프 부여 | 꾸미기에 소소한 기능성 버프 부여 (밸런스는 미세 조정 수준) |
| 구현 방향 | 허수아비 → 작물 성장 +3%, 연못 → 동물 행복도 +5% |

## 5. 재미 측정 지표 (KPI)

| 지표 | 목표 | 측정 방법 |
|------|------|-----------|
| 일일 접속 횟수 | 2~3회 | 루틴 체크 패턴 |
| 세션 길이 | 5~10분 | 접속 시간 측정 |
| 농장 확장률 | 월 2단계 이상 | 확장 이벤트 |
| 꾸미기 아이템 사용률 | 주 5개 이상 | 배치 변경 로그 |
| D7 리텐션 | 40% 이상 | 7일차 잔존율 |
